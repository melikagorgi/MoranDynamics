# Spatiotemporal Moran dynamics in continuous media

This repository contains MATLAB code used for the simulations and figures in the paper:

**Spatiotemporal Moran dynamics in continuous media**

## Files

```text
bd_1d.m          Birth-death update rule in a 1D cycle
db_1d.m          Death-birth update rule in a 1D cycle
fk_1d.m          Fisher-Kolmogorov comparison model

plot_Fig3.m      Script for generating Fig. 3
plot_Fig4A.m     Script for generating Fig. 4A
plot_Fig4B.m     Script for generating Fig. 4B
plot_Fig5A.m     Script for generating Fig. 5A
plot_Fig5B.m     Script for generating Fig. 5B
```

## Requirements

The code was written in MATLAB and uses standard MATLAB functions.
